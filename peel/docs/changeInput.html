Events

update

Parameters
  • e: the event
  • id: the id of the change
  • value: the value being set. {index: null/0...n, code:$string, size:'sm|md|lg', text:$string});
example
changeInput.on('changeinputupdate', function (e, id, value) { /****/ })

next

Parameters
  • e: the event
  • id: the id of the change
example
changeInput.on('changeinputnext', function (e, id) { /****/ })

prev

Parameters
  • e: the event
  • id: the id of the change
example
changeInput.on('changeinputprev', function (e, id) { /****/ })
**Note that all events are prefixed by the widget namespace 'changeinput'. e.g. 'update' event requires a 'changeinputupdate' listener.
